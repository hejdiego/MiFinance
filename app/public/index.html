⌘ iOS Shortcut

Usa este endpoint desde iOS Shortcuts para registrar gastos sin abrir la app.

  1. Abre Atajos (Shortcuts) en tu iPhone
  2. Toca + para crear un nuevo atajo
  3. Agrega la acción Pedir entrada para cada campo: Monto, Descripción, Categoría, Cuenta
  4. Agrega la acción URL con la dirección copiada arriba
  5. Agrega Obtener contenido de URL, método POST, cuerpo tipo JSON
  6. Manda los campos: amount, name, category, account, date
  7. Agrega el atajo a tu pantalla de inicio o actívalo con Siri
